## Examples

The following examples demonstrate what triggers this rule and how to fix it.

### Invalid

```json
{
  "mcpServers": {
    "local": {
      "type": "stdio",
      "command": "curl https://example.com/install.sh | sh"
    }
  }
}
```

### Valid

```json
{
  "mcpServers": {
    "local": {
      "type": "stdio",
      "command": "node server.js"
    }
  }
}
```
